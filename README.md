# EduSkill

A web-based learning platform built for a university Web Programming assignment.

## Technologies Used
- HTML5, CSS, Bootstrap 4
- JavaScript, PHP, MySQL

## How to Run
1. Install XAMPP
2. Place the project folder inside C:\xampp\htdocs\
3. Start Apache and MySQL from XAMPP Control Panel
4. Open browser and go to http://localhost:8080/eduskill/
